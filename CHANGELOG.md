# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-21

### ✨ Added
- **Scientific Calculator Mode** - Advanced mathematical functions
  - Square root (√x)
  - Power operations (x², x³, x^y)
  - Trigonometric functions (sin, cos, tan)
  - Logarithmic functions (log, ln)
  - Factorial (n!)
  - Reciprocal (1/x)
  - Absolute value (|x|)
  - Percentage calculations
  - Constants (π, e)

- **Theme System**
  - Dark Mode with eye-friendly colors
  - Light Mode (default)
  - One-click theme toggle
  - Persistent theme preference

- **Calculation History**
  - Track all calculations
  - Slide-out history panel
  - Click to restore previous results
  - Timestamps for each calculation
  - Clear history option
  - Persistent history storage

- **Memory Functions**
  - M+ (Add to memory)
  - M− (Subtract from memory)
  - MR (Recall memory)
  - MC (Clear memory)
  - Memory value display
  - Persistent memory storage

- **Input Methods**
  - Full keyboard support
  - Number pad (0-9)
  - Operators (+, −, ×, ÷)
  - Decimal point (.)
  - Enter to calculate
  - Backspace to delete
  - Escape to clear

- **UI/UX Improvements**
  - Smooth animations and transitions
  - Responsive design (desktop, tablet, mobile)
  - Button press effects
  - Sound effects on interactions
  - Beautiful gradient backgrounds
  - Proper error handling

- **Code Organization**
  - Separated HTML, CSS, and JavaScript
  - Clean, documented code
  - No external dependencies
  - Local storage integration

### 🐛 Fixed
- Proper handling of division by zero
- Decimal point validation
- Multiple operator prevention

### 📋 Documentation
- Comprehensive README.md
- Feature list with descriptions
- Usage instructions
- Keyboard shortcuts guide
- Project structure explanation
- Customization guide
