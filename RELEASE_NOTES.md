## 📦 Overview

Welcome to the **Advanced Calculator v1.0.0** - A powerful, feature-rich web-based calculator with scientific functions, dark mode, and intelligent history tracking!

This release brings a complete overhaul from the simple calculator to a professional-grade calculation tool suitable for students, professionals, and everyday users.

---

## ✨ Major Features

### 🧮 Scientific Calculator
- Full scientific functions library
- Trigonometric calculations (sin, cos, tan)
- Logarithmic functions (log, ln)
- Power operations (x², x³, x^y)
- Square root and factorial
- Percentage calculations
- Mathematical constants (π, e)

### 🌙 Theme System
- **Dark Mode** - Perfect for night time use
- **Light Mode** - Classic bright interface
- One-click theme switching
- Automatic theme preference saving

### 📜 Calculation History
- View all previous calculations
- Slide-out history panel
- Click any history item to restore result
- Timestamps for tracking
- One-click history clear

### 💾 Memory Functions
- **M+** - Add current value to memory
- **M−** - Subtract current value from memory
- **MR** - Recall stored memory value
- **MC** - Clear memory
- Real-time memory display

### ⌨️ Full Keyboard Support
Use your keyboard for faster calculations:
- `0-9` - Number input
- `+, −, ×, ÷` - Operators
- `.` - Decimal point
- `Enter` or `=` - Calculate
- `Backspace` - Delete last digit
- `Escape` - Clear display

### 📱 Responsive Design
- Desktop optimized
- Tablet friendly
- Mobile responsive
- Touch-friendly buttons
- Works on all modern browsers

### 🎨 Enhanced UI
- Beautiful gradient backgrounds
- Smooth button animations
- Sound effects on clicks
- Professional color scheme
- Intuitive button layout

## 🎯 How to Use

### Basic Mode
1. Click number buttons to enter values
2. Click operator buttons (+, −, ×, ÷)
3. Enter second number
4. Press `=` to calculate
5. Use `AC` to clear display

### Scientific Mode
1. Click "Scientific" button
2. Use advanced functions (√x, sin, cos, etc.)
3. Mix with basic operations
4. Switch back to Basic anytime

### Memory Operations
- **M+** - Adds current value to memory
- **M−** - Subtracts from memory
- **MR** - Shows memory value
- **MC** - Resets memory to 0

### History
- Click 📋 History button
- See all calculations with timestamps
- Click any item to use result
- Clear all with "Clear History"

### Theme
- Click 🌙/☀️ button (top-right)
- Toggle between dark and light modes
- Setting is automatically saved
